# 🏦 Smart Lender AI

An AI-powered Loan Approval Prediction System developed using Machine Learning and Flask.

---

## 📌 Project Overview

Smart Lender AI is a web-based machine learning application that predicts whether a loan application is likely to be approved based on applicant details such as income, education, credit history, employment status, loan amount, and property area.

The system helps automate the loan approval process by providing quick and data-driven predictions.

## ⚙ Installation

Clone the repository

```bash
git clone https://github.com/JinkalaSunitha/Smart-Lender-AI.git
```

Move into the project

```bash
cd Smart-Lender-AI
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app.py
```

Open your browser

```
http://127.0.0.1:5000
```
